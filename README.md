# mini-project

This is project use Java Spring boot 3

## Tech stack

- Build tool: maven >= 3.9.5
- Java: 17 or higher
- Framework: Spring boot 3.2.x
- DBMS: MySQL
- Deploy: Docker

## Prerequisites

- Java SDK 17 or higher
- A MySQL server

## Start application

`mvn spring-boot:run`

## Build application

`mvn clean package`
